# wxapp-servers

### Server For LeanEngine
  - WeChat

  - WeApp

### API

  - 快递

  - 上海天气

  - 上海实时公交

### 备注
  - 该程序部署在[leancloud](https://leancloud.cn) `(免费版本, 每天必须休眠6小时)
  `
  - 除 `实时到站信息` 外，其他信息均为自建 DB 存储
  - 因服务被其他程序使用， 现部署版本返回为 `mock数据`
  - 有问题欢迎提[issues](https://github.com/jmeviler/wxapp-server/issues)
